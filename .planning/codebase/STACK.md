## Key Dependencies

**Critical:**

- `community.sops` ^1.6.0 - Encrypted secrets management via SOPS/Age (`requirements.yml`)
- `@pulumi/pulumi` ^3.113.0 - IaC runtime (`infrastructure/package.json`)
- `@pulumi/digitalocean` 4.56.0 - DigitalOcean provider (`infrastructure/package.json`)

**Infrastructure:**

- Age - Encryption for SOPS (`tools/age/`)
- SOPS - Secrets management (`tools/sops/`, `.sops.yaml`)
- 1Password CLI - Service account token-based secret retrieval (`tools/1password_cli/`)

## Configuration

**Environment:**

- `~/.config/op/service-account-token` - 1Password service account token
- `~/.config/sops/age/keys.txt` - Age private key for SOPS decryption
- `.sops.yaml` - SOPS creation rules with Age public key
- `group_vars/all/defaults.yml` - Default variable values
- `group_vars/all/personal-info.sops.yml` - Encrypted personal information

**Build:**

- `ansible.cfg` - Ansible configuration (inventory, collections path, SOPS integration)
- `inventory.yml` - Remote host inventory
- `localhost.yml` - Local machine inventory (generated by bootstrap)
- `infrastructure/Pulumi.yaml` - Pulumi project config
- `infrastructure/Pulumi.dev.yaml` - Pulumi dev stack config
- `infrastructure/tsconfig.json` - TypeScript compiler options

## Platform Requirements

**Development:**

- macOS (Darwin) with Homebrew
- Debian/Ubuntu with apt
- Arch Linux with pacman/yay
- Python 3 interpreter at `/opt/homebrew/bin/python3` (macOS) or `/usr/bin/python3` (Linux)

**Production:**

- DigitalOcean Droplet (Debian 12, s-2vcpu-4gb, nyc1 region)
- SSH access via configured keys
